# ST (Suckless Terminal)

Personal build of ST

## Patches

- Blinking cursor
- Boxdraw
- Column redraw
- Desktop entry
- Glyph wide support (for boxdraw)
- Nano shortcuts
- Scrollback
    - Mouse scrollback
- Xresources
- Anysize

## Keybinds

|Bind|Action|
|-|-|
|Ctrl + k|Scroll up|
|Ctrl + j|Scroll down|
|Ctrl + Shift + K|Font +|
|Ctrl + Shift + J|Font -|
|Ctrl + Shift + H|Font size reset|
